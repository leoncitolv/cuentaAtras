Cómo usarla

1. Interfaz bonita

Subes este proyecto a GitHub Pages y lo abres desde Safari. Luego lo agregas a pantalla de inicio para que se vea como app.

2. Alertas reales

GitHub Actions revisa reminders.json cada 15 minutos y manda mensajes por Telegram.

3. Agregar recordatorios

Puedes crearlos aquí, exportar el JSON y reemplazar el archivo reminders.json en GitHub.

4. Seguridad

El token de Telegram jamás se pone en el HTML. Va en GitHub Secrets.
